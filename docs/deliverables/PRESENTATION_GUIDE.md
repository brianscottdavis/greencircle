# Presentation (PowerPoint) — Deliverable Guide

**Spec:** `docs/requirements/2026_Term 1 UX Project Spec.md` § Deliverables

## Required Content

The presentation must cover:

1. **Explanation of the waste management issue**
   - Local community context
   - Key problems (e.g. recycling contamination, landfill use, littering)
   - Why a digital platform can help

2. **Key UX features and user journeys**
   - Main features (dashboard, reporting, challenges, etc.)
   - Example user flows (e.g. resident reports contamination, joins a challenge)
   - How users move through the system

3. **How the design meets community needs**
   - Alignment with target audience (demographic, behavioural, psychometric, geographic)
   - Accessibility and inclusion
   - Evidence of user-centred design

## Suggested Structure

| Slide | Content |
|-------|---------|
| Title | Project name, team, date |
| Problem | Waste management issue in local community |
| Users | Target audience and needs |
| Features | Overview of core features (dashboard, reporting, guides, challenges, noticeboard, offline, admin) |
| User journeys | 2–3 example flows with screenshots from web prototype or wireframes |
| Design decisions | Why certain choices were made; UX principles applied |
| Community needs | How the design addresses accessibility, engagement, behaviour change |
| Summary | Key takeaways, success criteria met |
