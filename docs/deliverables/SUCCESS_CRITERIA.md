## Criteria Detail

1. **Targets local community members**
   - Content, language, and flows address residents, households, and local waste services
   - Not framed as a school project or student exercise

2. **UX design principles**
   - Usability, consistency, feedback, error prevention
   - Clear navigation and information hierarchy
   - Appropriate use of patterns (forms, dashboards, workflows)

3. **User-centred design thinking**
   - Evidence of user research or persona consideration
   - Design decisions justified by user needs
   - Iteration based on audience (demographic, behavioural, psychometric, geographic)

4. **Logical, accessible, realistic**
   - Flows make sense for the domain
   - Accessibility considered (readability, navigation, input methods)
   - Feasible for a civic/community context

5. **Computing Technology concepts**
   - Data representation, information systems, user interfaces
   - Links to syllabus or course outcomes where applicable
